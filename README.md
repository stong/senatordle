# Senatordle

Can you tell if a US Senator is Republican or Democrat just from their official portrait?

Play at [senatordle.com](https://senatordle.com)

This entire repo is vibe coded with Claude code except this sentence in the README. thx claudee <3

## About

Senatordle is a guessing game where you're shown the official portrait of a current US Senator and have to guess their political party. Test your political knowledge and see if you can spot the patterns!

Features:
- All 100 current US Senators
- Official portrait photos
- Score tracking and accuracy percentage
- Share your results
- Handles Independents who caucus with Democrats (Bernie Sanders, Angus King)

## Tech Stack

- [Next.js 16](https://nextjs.org) - React framework
- [React 19](https://react.dev) - UI library
- [TypeScript](https://www.typescriptlang.org) - Type safety
- [Tailwind CSS 4](https://tailwindcss.com) - Styling

## Running Locally

```bash
# Install dependencies
yarn install

# Run development server
yarn dev

# Build for production
yarn build
```

Open [http://localhost:3000](http://localhost:3000) to play locally.
